Guessing Game

Overview:

For this project, you will need to generate a random number. Then you have a user guess until they successfully find the random number. You will use the prompt() and alert() functions to get the input from the user and let them know how they are doing.

Generate the random number:

You will need to use the Math.random() function that is built into JavaScript. This function generates a random number between 0 (inclusive) and 1 (exclusive). In other words, a number between 0 and 0.9999999... You will also need to use the Math.floor(number) function, which takes the input number and returns that number rounded down to the nearest whole number. If you combine these two functions, you can generate a number between any two values (we'll call them min and max). You'll start by generating a random number, and multiplying that by (max - min + 1). Then you'll take that number and round it down using Math.floor() and add the min. In other words, you'll have the following.

let min = 3; let max = 10; let target = Math.random() * (max - min + 1); target = Math.floor(target) + min;
